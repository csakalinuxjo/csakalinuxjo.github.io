Ubuntu/Linux Mint
Please run the following commands from a terminal, one line after each other.
$ sudo add-apt-repository ppa:wine/wine-builds
$ sudo apt-get update
$ sudo apt-get install --install-recommends wine-staging
If you want to avoid typing the full path each time you can additionally install the winehq-staging package which provides compatibility symlinks:
$ sudo apt-get install winehq-staging



http://pipelight.net

Silverlight Ubuntu alatti telepítéséhez is fel kell rakni a pipelight-ot:

sudo add-apt-repository ppa:pipelight/stable
sudo apt-get update
sudo apt-get install --install-recommends pipelight-multi
sudo pipelight-plugin --update


A Silverlight esetében is az engedélyezés:

sudo pipelight-plugin --enable silverlight


Általános formula:
sudo pipelight-plugin --unlock pluginneve
sudo pipelight-plugin --enable pluginneve
